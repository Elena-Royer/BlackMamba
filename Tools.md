## Creating a simple class

- **With initialize() function**
```ruby
class iris:
    def initialize() -> None:
        self.color  = 'green'     # sheets color
        self.size   = 1.2         # its size
        self.width  = 0.3         # sheets width
    end:
    
    # get items
    def get( item : string ):
        return {color : color, size: size, width : width}
     end:
end:    
```
